PFSinatra
=======

A tiny web application to generate fibonacci series of <x> length or to generate
<y> passwords of <z> length

- **Requires**: ruby, sinatra 

        $ sudo gem install sinatra

- **Usage**:

        $ ruby main.rb

    Then visit `http://localhost:4567`
